# GoldenGym AI

## AI-Powered Fitness & Gym Management SaaS

GoldenGym AI is a modern fitness transformation and gym management platform built with React, TypeScript, Firebase, and AI-powered coaching.

### Features

* AI Workout Generator
* AI Diet Planner
* AI Fitness Coach
* Fitness Assessment Engine
* Analytics Dashboard
* Workout Tracking
* Water & Sleep Coach
* Body Measurements Tracking
* Transformation Roadmap
* Gamification & Achievements
* Community System
* Firebase Authentication
* Role-Based Access Control
* Responsive Mobile-First Design

### Tech Stack

#### Frontend

* React 19
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion
* Zustand
* React Router
* Recharts

#### Backend

* Node.js
* Firebase Authentication
* Firestore
* Firebase Storage
* Firebase Hosting

#### AI

* Gemini API (Current)
* Groq API (Planned Future Migration)

### Local Development

```bash
npm install
npm run dev
```

### Environment Variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=your_api_key
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
```
